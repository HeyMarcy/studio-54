# Sophia Williams Portfolio - Image Gallery

A responsive image gallery with lightbox functionality, optimized for performance.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file with your Cloudinary credentials:
   ```
   NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME='dwfhcqaat'
   CLOUDINARY_API_KEY='your-api-key'
   CLOUDINARY_API_SECRET='your-api-secret'
   CLOUDINARY_FOLDER='studio-54'
   ```

3. Upload images to your `studio-54` folder in Cloudinary

4. Fetch images from Cloudinary:
   ```bash
   npm run fetch-images
   ```

5. Build Tailwind CSS:
   ```bash
   npm run dev
   ```

6. Open `index.html` in your browser

## How It Works

Images are loaded dynamically from `dist/images.json`, which is generated by the fetch script. This keeps your credentials secure (server-side only) while allowing the gallery to update automatically.

```
Upload to Cloudinary → Run fetch-images → images.json → Gallery loads
```

## Adding New Images

1. Upload images to Cloudinary (`studio-54` folder)
2. Run `npm run fetch-images`
3. Refresh the page

That's it! The gallery will automatically display all images in your folder.

## Using with Next.js

If you're using Next.js, you can fetch images dynamically via an API route instead of the build script.

Copy `pages/api/gallery.js` to your Next.js project, then update your gallery component to fetch from `/api/gallery`:

```javascript
const response = await fetch('/api/gallery');
const images = await response.json();
```

The API route includes both Pages Router and App Router versions.

## Cloudinary Transformation Reference

| Parameter | Description | Example |
|-----------|-------------|---------|
| `w_` | Width | `w_600` |
| `h_` | Height | `h_400` |
| `c_fill` | Crop to fill (great for thumbnails) | |
| `c_fit` | Fit within bounds | |
| `c_limit` | Limit to max dimensions | |
| `q_auto` | Automatic quality optimization | |
| `f_auto` | Auto format (WebP/AVIF when supported) | |
| `ar_1:1` | Aspect ratio | |
| `g_auto` | Auto gravity (smart cropping) | |
| `e_blur:500` | Blur effect (for placeholders) | |

Example URL:
```
https://res.cloudinary.com/YOUR_CLOUD/image/upload/w_600,c_fill,q_auto,f_auto/gallery/photo-1
```

## Performance Features

- **Lazy loading**: Images below the fold load on scroll
- **Intersection Observer**: Efficient scroll-based animations
- **Responsive images**: Different sizes for different viewports
- **Auto format**: WebP/AVIF served when browser supports them
- **Reduced motion**: Respects `prefers-reduced-motion`
